PE Planning

We have produced, in partnership with a specialist IT Development Company, a web based lesson planning system.

The system is more advanced than any other Physical Education resource currently available to Primary Schools and because it is web based, it can be used anywhere with internet access.

Teachers can securely key in individual class information such as year group, NC level and their chosen PE topic. From just this small amount of information a high quality PE lesson plan will then be created with guidance and teaching tips. All plans can subsequently be edited in order to include differentiation and specific details that suit each individual class and school facility.

The system saves plans for future reference and will automatically save what has been taught in order to create Assessment for Learning reports at the end of each Unit.
